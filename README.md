 おみくじアプリ（恋愛バージョン）

 ■使い方
 ボタンを押すと、恋愛運がランダムに表示されます。

 恋みくじ（最強版）
ボタンをクリックするだけで、今日の恋愛運を占うことができるシンプルなWebアプリケーションです。

プロジェクトの概要
このプロジェクトは、HTML、CSS、JavaScriptを使用した初心者向けのWebアプリです。
ランダムに選ばれた運勢メッセージが表示され、ユーザーの行動を後押しします。

特徴
シンプル操作: 「おみくじを引く」ボタンを押すだけの簡単操作。

ランダム機能: JavaScriptの Math.random() を使用し、毎回異なる運勢を表示します。

レスポンシブなデザイン: シンプルで清潔感のある青を基調としたUI。

使用している技術
HTML5: アプリの構造定義。

CSS3: レイアウトとボタンのホバーエフェクトなどの装飾。

JavaScript (ES6): おみくじのロジックとDOM操作。

ファイル構成
Plaintext
.
├── index.html   # アプリの構造（UI）
├── style.css    # デザイン・スタイリング
└── script.js    # おみくじの実行ロジック
使い方
すべてのファイルを同じフォルダに保存します。

index.html をブラウザ（Chrome, Edge, Safariなど）で開きます。

「おみくじを引く」ボタンをクリックすると、結果が表示されます。

運勢の種類
おみくじには以下の4種類の結果が含まれています：

大吉: 素敵な出会いがあるかも！

中吉: 少し勇気を出すと良いことが起きる

小吉: 今日はゆっくり過ごそう

凶: 無理せず慎重に行動しよう

カスタマイズについて
script.js 内の fortunes 配列の内容を書き換えることで、独自のメッセージを追加したり変更したりすることが可能です。

JavaScript
const fortunes =[
    "大吉：最高の一日！",
    "吉：安定した運勢です",
    // ここに追加できます
];
